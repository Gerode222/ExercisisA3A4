Basic CREATE TABLE statement

The most basic CREATE TABLE statement creates a table with a few columns, defining what type of data is stored in each column.

CREATE TABLE customers (
    id INT,
    name VARCHAR(255),
    email VARCHAR(255)
);

Creating a table with a particular storage engine

MySQL provides a wealth of different storage engines with differing tradeoffs. The default, and most widely used is InnoDB. Other storage types offer specific behaviors. 

CREATE TABLE customers (
    id INT,
    name VARCHAR(255),
    email VARCHAR(255)
) ENGINE=MyISAM;

The same is going with `MEMORY`, `CSV` and many more. To understand the preferred method of defining the storage type, use https://dev.mysql.com/doc/refman/8.0/en/storage-engines.html You can inspect what table is set as default using:

SHOW VARIABLES LIKE 'storage_engine';
